Thursday Lecture

Guessing Game

GOALS:

For this assignment, we will learn the basics of working with user input, conditional statements and very simple loops. Also, as a part of this assignment, we will be using the "random" library from Python in order to call the randint() function.

Remember from class, this function can be called once you have imported random and then you can call the function by random.randint(< param1 >, < param2 >)

Requirements:
  1. Create a guessing game that generates a random number from 0 to the size that the user would like to play with. For input 10, numbers from 0 to 10 should be available in the game.
  2. Loop until the user answers correctly. You will need to take in new user input each time through the loop in order to check and see if the user got the answer correct.
  3. Give the user hints as to whether their guess was too high or too low relative to the secret number.
  4. Put an outer loop around the "guessing" portion of the game asking the user if they would like to start another game.
