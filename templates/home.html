K-State Codes' Python Class

K-State Codes offers a 0 credit introduction to programming class in python sponsored by the Knowledge Discovery in Databases Laboratory (KDD) at K-State.

This class is designed for beginners with no experience in programming, no prerequisites! K-State Codes welcomes students from all colleges in K-State to encourage connections and multidisciplinary research. This is a fast-paced course that will cover the basics in programming along some useful technologies used across academia such as:

  • File Handling
  • Web Scraping
  • OpenCV
  • SciPy
  • Machine Learning
